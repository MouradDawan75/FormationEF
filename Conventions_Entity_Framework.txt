Conventions de Entity Framework:

utiliser les annotations:
- Table name: [Table('t_products')]
- nom colonne: [Column("nom")] [MaxLength(256)]
- Clé primaire: si attribut contient le mot clé: Id -> EF l'utilise comme clé primaire - sinon utiliser l'annotation [Key]
- Type non nullable: [required]
- Champs unique: [index (isUnique = true)]
- clé étrangère: [foreingKey]




utiliser FluentApi: rédefinir la méthode onModelCreating() dans le context
